# Loan Approval Prediction Project

## Overview
This project predicts whether a loan will be approved (`Real`) or not (`Fake`) using machine learning. The dataset contains 4000 records with 19 variables, including numerical, categorical, and text features.

---

## Dataset Summary
* **Observations:** 4000  
* **Variables:** 19 (Text, Categorical, Numeric)  
* **No missing values or duplicates**  
* **Target variable:** `label` (Fake / Real)  

**Key features:**  
* Text: `title`, `text`  
* Categorical: `state`, `category`, `political_bias`, `fact_check_rating`, `is_satirical`  
* Numeric: `sentiment_score`, `word_count`, `readability_score`, `num_shares`, `num_comments`, `trust_score`, `source_reputation`, `clickbait_score`, `plagiarism_score`
